Componente: Correlation (src/apps/correlation)

Qué permite hacer en general
- Analizar relaciones entre variables numéricas usando tres tipos de vistas:
  - Correlation Matrix
  - Scatter Plot Matrix
  - PCA
- Definir una variable de grupo (categórica) para colorear puntos y segmentar análisis.
- Añadir múltiples vistas desde el panel.
- En cada vista: sincronizar/desincronizar con selección del Explorer, exportar SVG, exportar registros usados y cerrar la vista.

Panel lateral (control principal)
1) Analysis Context
- Seleccionar "Group variable" (variable categórica para segmentar/colorear).
- Ver estadísticas de contexto (AnalysisContextStats).

2) Chart Selection
- Seleccionar tipo de gráfico (Correlation Matrix, Scatter Plot Matrix, PCA).
- Añadir la vista seleccionada al grid.

Vistas disponibles en Correlation

A) Vista "Correlation Matrix"
Qué se puede hacer:
- Seleccionar variables incluidas (multiselección).
- Autoseleccionar variables con mayor correlación (Top correlations):
  - Definir N (nTop).
  - Botón para rellenar automáticamente variables más correlacionadas.
- Elegir método de correlación:
  - Pearson (r)
  - Spearman (rho)
  - Kendall (tau)
- Filtrar por rango de magnitud de correlación (slider |corr| min-max).
- Cambiar escala de color (varias paletas divergentes).
- Ajustar tamaño de etiquetas de ejes.
- Interacción visual:
  - Tooltip por celda con par de variables y valor de correlación.
  - Resaltado de celda al pasar el ratón.
- Información adicional:
  - Muestra en "Info" el ranking top-N de correlaciones (si nTop > 0).

Notas funcionales:
- Requiere al menos 2 variables.
- Si hay valores inválidos en las variables seleccionadas, la vista lanza error de cálculo.

B) Vista "Scatter Plot Matrix"
Qué se puede hacer:
- Seleccionar variables incluidas (multiselección).
- Ajustar puntos:
  - Tamaño.
  - Opacidad.
- Mostrar/ocultar leyenda.
- Ajustar tamaño de etiquetas de ejes.
- Interacciones dentro de la matriz:
  - Tooltip en puntos (grupo, valores de ambas variables y ID si existe).
  - Brush rectangular por celda para resaltar subconjuntos.
  - Ctrl + click sobre una celda para abrir "big scatter" (vista ampliada de ese par).
  - En vista ampliada: Ctrl + click para volver a la matriz completa.
- Interacciones de leyenda por grupo:
  - Hover en un grupo: aislamiento temporal de ese grupo.
  - Click en etiqueta: ocultar/mostrar grupo.
  - Click en círculo: blur/unblur de grupo.

Notas funcionales:
- Requiere al menos 2 variables para dibujar.
- Usa la Group variable para color por clase.

C) Vista "PCA · <n> Variables"
Qué se puede hacer (PCA base):
- Seleccionar variables incluidas (multiselección).
- Botón "Add all visible" para añadir todas las variables visibles del dataframe.
- Ajustar puntos:
  - Tamaño.
  - Opacidad.
- Mostrar/ocultar leyenda.
- Ajustar tamaño de etiquetas.
- Interacciones de gráfico:
  - Tooltip por punto (grupo, PC1, PC2 e ID si existe).
  - Zoom y pan (rueda/arrastre).
  - Leyenda con aislamiento/ocultación temporal por grupos.

Modo Lasso en PCA (funcionalidad avanzada):
- Activar/desactivar "Lasso mode" desde settings.
- Al activar:
  - Pide nombre de nueva columna categórica para guardar grupos (target column).
  - No permite usar nombre ya existente.
- Panel flotante "PCA Lasso Groups" permite:
  - Elegir acción de selección: Add / Remove.
  - Añadir grupos.
  - Renombrar grupos.
  - Elegir grupo activo.
  - Limpiar asignaciones.
  - Definir nombre del grupo "Unassigned".
  - Ver contador n de puntos por grupo y no asignados visibles.
  - Guardar resultado en dataset (Save to dataset).
  - Descargar preview en CSV (Download CSV).
- Interacciones de asignación:
  - Click izquierdo sobre punto: asigna/quita punto al grupo activo.
  - Click derecho + arrastre: lasso libre para selección múltiple.
  - Pan se mantiene con arrastre izquierdo.

Persistencia al guardar lasso:
- Inserta/actualiza la columna categórica en el dataframe.
- Añade la columna a navioColumns y varTypes.
- Inserta/actualiza el nodo en metadata/hierarchy con descripción de estratificación PCA.

Notas funcionales:
- PCA requiere al menos 2 variables válidas.
- Variables con valores inválidos se marcan en rojo en settings y quedan excluidas del cálculo PCA.
- Al seleccionar una variable inválida, muestra aviso de exclusión.

Acciones comunes disponibles en todas las vistas
- Sync on/off con selección del Explorer.
- Exportar registros usados en la vista (View records export).
- Descargar gráfico SVG.
- Abrir panel de Settings.
- Ver Info (cuando la vista provee contenido).
- Cerrar vista.

Resumen rápido de cuándo usar cada vista
- Correlation Matrix: para visión global de fuerza/signo de correlaciones entre muchas variables.
- Scatter Plot Matrix: para inspección visual de forma de relación, outliers y segmentación por grupo.
- PCA: para reducir dimensión, explorar estructura global y crear estratificaciones manuales con lasso.
